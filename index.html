Data Flow: Market data is ingested from the IBKR paper environment into Kafka event streams. The AI signal engine processes this data to generate simulated trading signals, which are routed through the IBKR paper gateway for execution modeling. Redis maintains real-time state and positions, while Elasticsearch captures logs and metrics for monitoring, replay, and analysis.
